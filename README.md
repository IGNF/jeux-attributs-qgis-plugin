# IGN_jeux_attributs

This plugin provides assistance for editing attribute values across different fields.

## Pré-requis

Mandatory : The "plugin\_maitre" must be installed.
Link to the plugin maître : [maitre-qgis-plugin sur GitHub](https://github.com/IGNF/maitre-qgis-plugin)

## Fonctionnalités

French : 
* Modification des attributs des entités par simple clic. 

English :
* Edit feature attributes with a single click.
